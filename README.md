Dev Tracker – Frontend

The Dev Tracker Frontend is a modern React-based dashboard designed to help developers track productivity, manage tasks, monitor coding activity, and visualize performance insights.
It connects with the Dev Tracker API to provide a complete productivity management platform including Kanban task boards, time tracking, analytics dashboards, and GitHub activity insights.
This project focuses on clean UI architecture, scalable frontend structure, and real-time productivity insights for developers.

🛠 Tech Stack

React Vite
Tailwind CSS
React Router
Axios
Recharts (for analytics charts)
